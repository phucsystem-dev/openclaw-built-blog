About This Dashboard

This dashboard visualizes historical Vietnam gold price data collected through an automated system:

  • Data Source: Daily collection via GitHub Actions at 7 AM Melbourne time
  • Storage: Markdown files in the repository's data/ directory
  • Brands: SJC, DOJI, PNJ (major Vietnamese gold brands)
  • Cities: Ho Chi Minh City and Hanoi
  • Time Scales: View 1 week, 2 weeks, 1 month, 3 months, 6 months, or 1 year of data
  • Export: Download data as CSV or JSON for analysis

For real-time API integration, configure API keys from VNAppMob or MetalPriceAPI in the GitHub Actions workflow.
